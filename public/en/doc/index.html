REST

Base url:

https://dragon-ball-super-api.herokuapp.com/api

Url chứa thông tin về các tài nguyên có sẵn của API. Tất cả các yêu cầu sẽ trở lại json data.

GET https://dragon-ball-super-api.herokuapp.com/api
                        
{
    "characters": "https://dragon-ball-super-api.herokuapp.com/api/characters"
}

Tài nguyên có sẵn:

  • Characters: get all characters.

Thông Tin

Tài liệu này sẽ giúp bạn làm quen với các tài nguyên API Dragon Ball bằng cách hiển thị cho bạn các truy vấn daa khác nhau.

Sơ lược nhân vật

Key Type Description
id int Id của nhân vật
imageUrl string (url) Một hình ảnh (.png) của nhân vật
name string Tên nhân vật
originplanet string Hành tinh gốc của nhân vật
role string Vai trò hoặc chức năng quan trọng nhất của nhân vật
specie string Hành tinh của nhân vật
status string Trạng thái của nhân vật
transform array Biến đổi nhân vật (nếu có)
universe string Vũ trụ mà nhân vật thuộc về

Lấy tất cả nhân vật

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters

You can access the list of all characters using the endpoint /characters.

GET https://dragon-ball-super-api.herokuapp.com/api/characters
                        
[
    {
        "id": 124,
        "name": "Daishinkan",
        "specie": "Angel",
        "role": "Sumo Sacerdote",
        "imageUrl": "https://i.ibb.co/cY94Ddp/Daishinkan.png",
        "status": "Vivo",
        "universe": "0",
        "transform": [],
        "originplanet": "Desconocido"
    },
    ...
]

Phân trang đơn giản

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters?limit={max-characters}

Bạn có thể truy cập một phân trang nhất định bằng tham số limit /characters?limit={max-characters}. This will return an array with the characters separated based on the parameter limit

GET https://dragon-ball-super-api.herokuapp.com/api/characters/characters?limit=50
                        
[
    0: {...}
    1: {...}
    2: {...}
]
                        
// Array of 50 characters each position.

Offset y (giới hạn)

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters?offset={x}?limit={y}

Bạn có thể truy cập vào một phân trang nhất định bằng cách sử dụng các tham số truy vấn offset y limit /characters?offset={x}?limit={y}.

GET https://dragon-ball-super-api.herokuapp.com/api/characters/characters?offset=0?limit=50
                        
[
    0: {...}
    1: {...}
    2: {...}
]

Tìm kiếm tên nhân vật

Endpoint:

https://dragon-ball-super-api.herokuapp.com/api/characters/{name}

Bạn có thể lấy một ký tự bằng cách sử dụng tên /characters/{name}.

GET https://dragon-ball-super-api.herokuapp.com/api/characters/Gohan
                        
{
    "id": x,
    "imageUrl": "https://i.ibb.co/XjmRwZp/Dragon-Ball-Gohan-PNG-Pic.png",
